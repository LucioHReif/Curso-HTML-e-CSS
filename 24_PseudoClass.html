PSEUDO CLASSE Uma pseudoclasse CSS é uma palavra-chave adicionada a um seletor que especifica um estado especial do(s) elemento(s) selecionado(s). Por exemplo, a pseudoclasse pode ser usada para selecionar um botão quando o ponteiro do usuário passa sobre o botão e esse botão selecionado pode ser estilizado. /* Any button over which the user's pointer is hovering */ button:hover { color: blue; } Uma pseudoclasse consiste em dois pontos ( :) seguidos pelo nome da pseudoclasse (por exemplo, :hover). Uma pseudoclasse funcional também contém um par de parênteses para definir os argumentos (por exemplo, :dir()). O elemento ao qual uma pseudoclasse está anexada é definido como um elemento âncora (exemplo: button:hover). Pseudo-classes permitem aplicar um estilo a um elemento não apenas em relação ao conteúdo da árvore do documento, mas também em relação a fatores externos como o histórico do navegador ( , por exemplo), o status :visitedde seu conteúdo (como :checkedem certos elementos de formulário) ou a posição do mouse (como :hover, que permite saber se o mouse está sobre um elemento ou não).
