PROPRIEDADE CONTENT A contentpropriedade é usada com os pseudo-elementos ::before e ::after , para inserir o conteúdo gerado. ** Sintaxe - content: normal|none|counter|attr|string|open-quote|close-quote|no-open-quote|no-close-quote|url|initial|inherit; Value Description normal Valor padrão. Define o conteúdo, se especificado, como normal, cujo padrão é "none" (que não é nada) none Define o conteúdo, se especificado, como nada counter Define o conteúdo como um contador attr(attribute) Define o conteúdo como um dos atributos do seletor string Define o conteúdo para o texto que você especificar open-quote Define o conteúdo como uma citação de abertura close-quote Define o conteúdo como uma aspa de fechamento no-open-quote Remove a citação de abertura do conteúdo, se especificado no-close-quote Remove a aspa de fechamento do conteúdo, se especificado url(url) Define o conteúdo como algum tipo de mídia (uma imagem, um som, um vídeo, etc.) initial Define esta propriedade com seu valor padrão. Leia sobre inicial inherit Herda esta propriedade de seu elemento pai. Leia sobre herança
