IFRAME HTML Um iframe HTML é usado para exibir uma página da web dentro de uma página da web. A tag HTML -> Definir altura e largura Use os atributos heighte widthpara especificar o tamanho do iframe. A altura e a largura são especificadas em pixels por padrão: Exemplo Ou você pode adicionar o styleatributo e usar o CSS heighte widthas propriedades: -> Remova a borda Por padrão, um iframe tem uma borda ao seu redor.
