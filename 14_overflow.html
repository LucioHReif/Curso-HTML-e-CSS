PROPRIEDADE OVERFLOW A overflow propriedade especifica se deve cortar o conteúdo ou adicionar barras de rolagem quando o conteúdo de um elemento for muito grande para caber na área especificada. A overflow propriedade tem os seguintes valores: -> visible: Padrão. O estouro não é cortado. O conteúdo renderiza fora da caixa do elemento -> hidden: O estouro é cortado e o restante do conteúdo ficará invisível -> scroll: O estouro é cortado e uma barra de rolagem é adicionada para ver o restante do conteúdo -> auto: Semelhante ao scroll, mas adiciona barras de rolagem somente quando necessário Nota: A overflowpropriedade só funciona para elementos de bloco com uma altura especificada.
