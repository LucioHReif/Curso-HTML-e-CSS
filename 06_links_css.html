PROPRIEDADES DE LISTA EM CSS - list-style-image: define uma imagem como marcador da lista. Valores: none; URL: url(caminho/marcador.gif) - list-style-position: define a posição do marcador de lista; outside: marcador fora do alinhamento do texto inside: marcador alinhado com texto - list-style-type: define o tipo do marcador da lista; none: sem marcador disc: círculo (bolinha cheia) circle: circunferência (bolinha vazia) square: quadrado cheio decimal: números 1, 2, 3, 4, ... decimal-leading-zero: decimal com 0 no início lower-roman: romano minúsculo i, ii, iii, iv, ... upper-roman: romano maiúsculo I, II, III, IV, ... lower-alpha: letra minúscula a, b, c, d, ... upper-alpha: letra maiúscula A, B, C, D, ... lower-greek lower-latin upper-latin hebrew armenian georgian cjk-ideographic hiragana katakana hiragana-iroha katakana-iroha - list-style: maneira abreviada para definir DUAS propriedades.
