Web Design Fundamentos: Aprenda HTML CSS e Javascript PICTURE HTML A tag oferece aos desenvolvedores da Web mais flexibilidade na especificação de recursos de imagem. O uso mais comum do elemento será para direção de arte em designs responsivos. Em vez de ter uma imagem dimensionada para cima ou para baixo com base na largura da janela de visualização, várias imagens podem ser projetadas para preencher melhor a janela de visualização do navegador. O elemento contém duas tags: uma ou mais tags e uma tag . O navegador procurará o primeiro elemento em que a consulta de mídia corresponda à largura atual da viewport e exibirá a imagem adequada (especificada no atributo srcset).
